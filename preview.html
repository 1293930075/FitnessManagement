高碳日 碳水 310-340g / 脂肪 50g

配大肌群训练（腿、背）。午餐必须吃足，糙米饭200g+。训后30min内补红薯150g+鸡胸150g。高碳不等于放开吃，脂肪要压住。

中碳日 碳水 190-220g / 脂肪 62g

配次要肌群（肩、腹）。午餐糙米饭120g，训后红薯100g。脂肪来源可以稍微宽松，橄榄油、鸡蛋正常吃。

低碳日 碳水 80-100g / 脂肪 70g

配胸、三头或休息日。脂肪从好来源补：牛油果、鸡蛋、橄榄油、三文鱼。训练日训后仍补50g红薯，休息日不补。

蛋白质 所有碳日固定 185g，分5-6餐

不喝蛋白粉，每餐必须有蛋白质来源。参考：鸡胸250g(78g蛋白) + 三文鱼150g(30g) + 鸡蛋5个(30g) + 豆腐/豆干100g(12-15g) + 虾100g(18g) ≈ 183g。

高碳日优先保证午餐和训后补给，中碳日强调训练质量与脂肪平衡，低碳日则把恢复、控脂和蛋白质摄入放在最前面。

蛋白质来源（无乳糖）

主力补给与便携高蛋白选择
鸡胸肉31g蛋白 / 100g。主力来源，每天250-300g，水煮/气炸/煎均可，批量煮好冷藏3天。
鸡蛋6g蛋白 / 个。每天4-5个，高碳日可全蛋，低碳日2全蛋+2蛋白，乳糖不耐受可放心吃。
三文鱼20g蛋白 / 100g + Omega-3。抗炎降脂，每周3次，煎/烤/生食均可，晚餐首选。
金枪鱼罐头25g蛋白 / 100g。水浸版，低脂高蛋白，工作日加餐救急神器，开罐即食。
虾18g蛋白 / 100g。低脂，低碳日最适合，白灼最方便。
瘦牛肉26g蛋白 / 100g。里脊/腱子，富含铁锌，每周1-2次，高碳日优先。
北豆腐/豆干12-15g蛋白 / 100g。植物蛋白补充，睡前餐或加餐，消化慢利于夜间合成。

碳水来源（净碳慢消化）

主食、训前、训后补给分工明确
糙米GI 55，主食首选，高碳日每餐熟重150-200g。
燕麦早餐主力，饱腹感强，干重60-80g，所有碳日均可用。
红薯训后首选，GI 58，富含钾，高碳日150g / 低碳日50g。
香蕉/椰枣训前快速补糖原，工作日出发前路上吃，不算正餐。
藜麦含蛋白质，中碳日代替糙米，口感多样。

好脂肪来源（低碳日重点）

用来稳定激素与饱腹感
牛油果低碳日半个，单不饱和脂肪，配鸡蛋早餐最佳。
橄榄油每天10-15ml，炒菜或拌沙拉，抗炎。
坚果工作日下午少量（10-15粒核桃/杏仁），不方便吃饭时应急用。

工作日执行要点

午餐
训前最重要的一餐，不能省。高碳日午餐糙米200g，这是晚上6点训练的主要能量来源，距离训练6小时，吃饱没问题。
17:30
出发前小补（不算加餐）。香蕉1根或椰枣3-4颗，路上吃，15分钟内消化，给训练快速补一点糖原。低碳日可吃少量坚果代替。
训后
30分钟内必须吃。训练结束20:00-20:30，回家或路上准备好，这餐不能拖到21点后。可以提前备好鸡胸肉和红薯，微波炉加热2分钟搞定。
睡前
21:30-22:00，慢蛋白补充。鸡蛋2个+豆干/豆腐，慢消化蛋白，支撑夜间肌肉合成。不要吃碳水。

进度调整信号

体重停滞
连续2周无变化 → 先加有氧5-10分钟，再考虑低碳日减碳水20-30g，两个不同时动。
掉重太快
每周超1kg → 高碳日加碳水50g，防掉肌，体重数字不是唯一指标。
力量下降
训练重量明显掉 → 高碳日加量，不是你变弱，是碳水不够。
4周评估
同时间同光线拍腹肌对比照，视觉进步比体重更准确。体重没动但视觉改善 = 增肌同时减脂，是最理想的状态。
周末把鸡胸肉一次煮 500-600g、红薯蒸好冷藏，工作日训后直接加热就能吃，这是这个方案真正能长期坚持下去的关键。
